# Baccarat-Practice-Game
A lightweight Baccarat-style practice game built purely with HTML, CSS, and JavaScript. Players choose a bet amount, place a bet on PLAYER, BANK, or TIE, and the game simulates card draws and round resolution while tracking cumulative revenue (net wins and losses).
Key Features:
Pure front-end implementation using only HTML, CSS, and JavaScript.
Fixed bet amounts: 10, 50, 100, 500, 1000.
Bet targets: PLAYER, BANK, TIE.
Card dealing simulation with simple baccarat-style sum logic.
Real-time UI showing cards, hand sums, messages, and a running revenue counter that reflects net gains and losses.
Simple state handling for rounds, bets, and payouts.

How to Play
Select a bet amount using the bet buttons.
Choose where to place the bet: PLAYER, BANK, or TIE.
Click to draw cards; the game displays each hand’s cards and sums.
If rules require, draw an additional card for player or bank.
The result updates revenue: winning adds the bet amount (or the tie payout if applicable), losing subtracts the bet amount.

Tech Stack and Installation
Technologies
HTML for structure
CSS for styling
JavaScript for game logic and UI updates

Install and Run
 Clone the repository.
Open index.html in any modern browser.
No build tools or server required.
